## 文件结构

```shell
.
├── config              // 构建配置
├── public              // 不参与构建的静态文件
├── scripts             // 构建脚本
├── src                 
│   ├── assets          // 资源文件
│   ├── components      // 通用组件
│   ├── layouts         // 框架布局组件
│   ├── models          // 模块封装，基于redux
│   ├── pages           // 页面组件
│   ├── router          // 路由
│   ├── services        // api
│   ├── utils           // 基础组件、工具
│   │   ├── axios       // axios
│   │   ├── redux       // redux 封装 ????
│   │   └── utils       // 基础工具
│   ├── App.css         // 根组件样式
│   ├── App.js          // 根组件
│   ├── index.css       // 全局样式 慎用
│   └── index.js        // 项目入口
├── package.json
├── README.md
└── yarn.lock
```
