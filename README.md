## 文件结构

```shell
.
├── config              // 构建配置
├── public              // 不参与构建的静态文件
├── scripts             // 构建脚本
├── src                 
│   ├── api             // 请求管理
│   ├── assets          // 资源文件
│   ├── components      // 通用组件
│   ├── layouts         // 框架布局组件
│   ├── pages           // 页面组件
│   ├── router          // 路由
│   ├── services        // api
│   ├── store           // redux
│   ├── utils           // 基础组件、工具
│   │   ├── auth        // 登陆权限相关函数
│   │   ├── redux       // request axios
│   │   └── tools       // 业务工具
│   ├── App.css         // 根组件样式
│   ├── App.js          // 根组件
│   └── index.js        // 项目入口
├── package.json
├── README.md
└── yarn.lock
```
